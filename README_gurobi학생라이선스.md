## 결론: 학교 이메일 + 대학 네트워크(또는 VPN)로 무제한 라이선스 무료 발급

---

## 📋 신청 절차 (Named-User 라이선스 기준)

재학 중인 인가 대학의 학생·교직원이면 누구나 무료로 발급받을 수 있으며, 수업·연구 등 비상업적 목적에만 사용 가능합니다.

### Step 1. 계정 생성
**https://portal.gurobi.com** 에서 회원가입 시 "Academic" 을 선택하고, 이름·소속 대학·국가를 입력한 뒤 "Access Now"를 클릭합니다.

> **중요**: 반드시 **학교 이메일 주소** (예: `@unist.ac.kr`, `@kaist.ac.kr`)로 가입해야 합니다.

### Step 2. 라이선스 신청
포털 로그인 → **Licenses** 탭 → **Request License** → `Academic Named-User` 또는 `Academic WLS` 선택

### Step 3. 활성화
활성화 시에는 **대학 네트워크에 연결된 상태**여야 합니다. 활성화 이후에는 오프라인 환경에서도 사용 가능합니다.

터미널에서:
```bash
grbgetkey [발급받은 라이선스 키]
```

---

## 🆚 라이선스 종류 비교

| | Named-User | WLS (Web License Service) |
|---|---|---|
| 특징 | 1기기 고정 | 기기 이동 가능 |
| 컨테이너 | ❌ Docker 불가 | ✅ Docker/클라우드 OK |
| 활성화 | 대학 네트워크 1회 | 동일 |
| 추천 대상 | 개인 노트북 고정 사용 | 연구 서버·여러 환경 |
| 유효기간 | **1년 (매년 재신청)** | 동일 |

만료 후에는 갱신이 아니라 **새 라이선스를 재신청**하는 방식으로 계속 사용할 수 있으며, 재학 중인 한 반복 신청 가능합니다.

---

## ⚠️ 주의사항 & 시사점

**집에서 신청할 때** 대학 네트워크가 아니면 활성화가 안 됩니다. 학교 Wi-Fi에 접속하거나 학교 VPN을 사용해서 활성화 명령어를 실행해야 합니다.

졸업 후에도 **Take Gurobi With You(TGWY) 프로그램**을 통해 졸업 24개월 이내 직장인도 무제한 라이선스를 계속 사용할 수 있습니다. 졸업 전 미리 신청해 두는 것이 좋습니다.
